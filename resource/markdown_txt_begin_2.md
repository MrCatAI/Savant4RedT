```plaintext

+ 涉及内容属性：?
+ 安全等级划分：[低、中或者高?]危险等级
+ 划分理由：

## 具体内容分析

### 分析点一

截取目标内容：⌈⌋
分析结果：

### 分析点二

截取目标内容：⌈⌋
分析结果：

### 分析点三

截取目标内容：⌈⌋
分析结果：
```

### 🍎 输入格式

```python
# 你想分析的内容
context = ""

# 输入格式
Input = f"""请你分析以下内容:
'''
{context}
'''
输出分析结果
"""
```
